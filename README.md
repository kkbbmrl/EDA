# EDA Mini Project (ECMD Module)

This repository contains the Exploratory Data Analysis (EDA) mini project for the ECMD module. It includes datasets, notebooks, and scripts used to explore, visualize, and summarize key insights.

- Overview: Perform data cleaning, descriptive statistics, and visualizations.
- Contents: `data/`, `notebooks/`, `reports/`, `src/`.
- Getting Started: Open the notebooks in Jupyter and run cells in order.
- Requirements: Python 3.10+, pandas, numpy, matplotlib, seaborn.

Feel free to contribute improvements or open issues for feedback.
